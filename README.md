# mypaint

A minimal MS Paint–style drawing application for Linux, built with PySide6.

Created with vibe coding and love.

Open for use, modification, and inspiration.

## Features

- Brush, pencil, line, curve, rectangle, ellipse, polygon, arrow, text, and selection tools
- Fill and stroke style controls
- Zoom with pixel grid
- Color palette with color 1 / color 2
- Undo / redo
- Transparent selection mode
- PNG / BMP / JPEG export
